¿Cómo usar condicionales?
Sintaxis:
{{#si VARIABLE OPERADOR VALOR}}texto{{/si}}
{{#si VARIABLE OPERADOR VALOR}}texto si cumple{{sino}}texto si no{{/si}}

Operadores: < > <= >= == !=

Ejemplos:
{{#si edad < 18}}Necesitas autorización de tus padres.{{sino}}No necesitas autorización.{{/si}}
{{#si avance < 30}}⚠️ Tu avance está por debajo del 30%.{{/si}}
{{#si estado == activo}}Tu matrícula está activa.{{sino}}Contacta a soporte.{{/si}}
{{#si dias_disponibles <= 7}}¡Solo te quedan {{dias_disponibles}} días!{{/si}}
